# @saber2pr/serve

> http file server.

```bash
# from npm
npm install @saber2pr/serve -g

# from github
git clone https://github.com/Saber2pr/serve.git
```

# Usage

```bash
serve # default port 5000

serve 3000 # use port 5000

# quick start
npx @saber2pr/serve
```

---

## start

```bash
npm install
```

```bash
npm start

npm run dev

```
